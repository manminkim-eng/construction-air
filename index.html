비작업일수 산정 공식 (고시 제8~10조)
비작업일수 = A + B − C
C (중복일수) = (A × B) ÷ 달력일수
A = 기상조건 비작업일수 (혹서·동절기·강우·강풍·강설·미세먼지 합산)
B = 법정공휴일수 (관공서 공휴일에 관한 규정 적용)
C = 중복일수 (소수점 첫째자리 반올림)
최솟값: 월 8일 보장 (주 40시간 근무제 하한선)
